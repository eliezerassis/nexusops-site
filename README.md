# Nexus Ops — Site de Apresentação

Site de marketing para o **Nexus Ops**, plataforma de gestão operacional para empresas de serviços (limpeza, hospitalidade, manutenção, condomínios).

![Screenshot do site](public/screenshot.png)

## Stack

- [Next.js 16](https://nextjs.org) (App Router)
- TypeScript
- Tailwind CSS
- [lucide-react](https://lucide.dev)

## Seções

- **Navbar** — navegação com menu mobile
- **Hero** — headline, CTAs e badge de confiança
- **Features** — 6 módulos da plataforma
- **Como Funciona** — 3 passos numerados
- **Segmentos** — 4 setores atendidos
- **Pricing** — planos Basic, Pro e Enterprise
- **FAQ** — accordion com 6 perguntas
- **CTA** — seção final de conversão
- **Footer** — links e GitHub

## Rodando localmente

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no browser.

## Deploy

Hospedado na Vercel com autodeploy via GitHub.
